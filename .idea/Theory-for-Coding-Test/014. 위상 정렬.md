## 1. 위상 정렬 (Topological Sort)

- 사이클이 없는 방향 그래프(DAG: Directed Acyclic Graph)의 모든 노드를 방향성을 거스르지 않도록 나열하는 정렬 방법

### 위상 정렬의 기본 개념

- 방향 그래프에서 정점들을 방향을 거스르지 않도록 나열하는 것 
- 진입 차수(Indegree)를 기반으로 동작 
- 사이클이 없어야 수행 가능

### 동작 과정

#### 큐를 이용하는 방법

1. 진입 차수가 0인 노드를 큐에 삽입 
2. 큐가 빌 때까지 다음을 반복:
   1. 큐에서 노드를 꺼내고 결과 리스트에 추가 
   2. 해당 노드에서 나가는 간선 제거 (즉, 연결된 노드의 진입 차수 감소)
   3. 새롭게 진입 차수가 0이 된 노드를 큐에 삽입
3. 결과 리스트가 위상 정렬의 결과(큐에 들어온 노드의 순서가 결과 리스트)

#### DFS와 스택을 이용하는 방법

1. 그래프 구성
   - 방향 그래프이므로 나가는 간선만 기록(진출차선만 기록)
2. 시작 노드(S) 설정
   - 진입 간선이 없는 노드를 시작점으로 선택 (예: node[0])
3. DFS 수행
   - 방문하지 않은 인접 노드로 재귀적으로 DFS 실행 
   - 인접 노드가 더 이상 없으면 현재 노드를 스택에 저장
4. 정렬 결과 출력
   - DFS가 끝나면 스택에서 하나씩 꺼내 출력 
   - 출력 순서가 위상 정렬 결과

---

## 2. 특징

- DAG(Directed Acyclic Graph, 방향이 있고 사이클이 없는 그래프)에서만 가능 
- 여러 개의 정렬 결과가 존재할 수 있음 (큐에 동시에 여러 노드가 들어가는 경우)
- 모든 노드를 방문하기 전에 큐가 비면 → 사이클 존재 
- DFS를 활용한 위상 정렬도 가능 (스택 활용)

## 3. 구현

```java
// 큐를 이용한 구현
import java.util.*;

public class Main {

    public static int v, e;
    public static int[] indegree;
    public static List<List<Integer>> graph = new ArrayList<>();

    public static void topologySort() {
        ArrayList<Integer> result = new ArrayList<>();
        Queue<Integer> q = new LinkedList<>();

        for (int i = 1; i <= v; i++) {
            if (indegree[i] == 0) {
                q.offer(i);
            }
        }

        while (!q.isEmpty()) {
            int now = q.poll();
            result.add(now);
            for (int i = 0; i < graph.get(now).size(); i++) {
                int next = graph.get(now).get(i);
                indegree[next] -= 1;
                if (indegree[next] == 0) {
                    q.offer(next);
                }
            }
        }

        for (int node : result) {
            System.out.print(node + " ");
        }
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        v = sc.nextInt();
        e = sc.nextInt();
        
        indegree = new int[v + 1];

        for (int i = 0; i <= v; i++) {
            graph.add(new ArrayList<>());
        }

        for (int i = 0; i < e; i++) {
            int a = sc.nextInt();
            int b = sc.nextInt();
            graph.get(a).add(b);
            indegree[b] += 1;
        }

        topologySort();
    }
}
```

import java.util.*;

public class Main {

    static int v; // 노드 수
    static int e; // 간선 수
    static List<List<Integer>> graph = new ArrayList<>();
    static boolean[] visited;
    static Stack<Integer> stack = new Stack<>();

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        v = sc.nextInt();
        e = sc.nextInt();

        visited = new boolean[v + 1];
        
        // 그래프 초기화
        for (int i = 0; i <= v; i++) {
            graph.add(new ArrayList<>());
        }

        // 간선 추가 (방향 그래프)
        for (int i = 0; i < e; i++) {
            int a = sc.nextInt();
            int b = sc.nextInt();
            graph.get(a).add(b);
        }

        // 위상 정렬 시작
        for (int i = 1; i <= v; i++) {
            if (!visited[i]) {
                topologicalSort(i);
            }
        }

        // 결과 출력
        System.out.println("위상 정렬 순서");
        while (!stack.isEmpty()) {
            System.out.print(stack.pop() + " ");
        }
    }

    static void topologicalSort(int current) {
        visited[current] = true;
        for (int next : graph.get(current)) {
            if (!visited[next]) {
                dfs(next);
            }
        }
        stack.push(current);
    }
}

### 4. 시간 복잡도

- O(V + E)
- 노드 개수(V), 간선 개수(E)에 비례하여 매우 효율적임
